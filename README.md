## Snake Game

This is a classic **Snake Game** built using simple and clean code. The player controls a snake that moves around the screen, eats food, and grows longer. The goal is to score as many points as possible without hitting the walls or the snake’s own body.

### Features

* Smooth snake movement
* Food spawns at random positions
* Score increases when food is eaten
* Snake grows longer with each food
* Game-over when snake hits wall or itself

### How to Play

* Use **Arrow Keys (↑ ↓ ← →)** to move the snake
* Eat the red food to grow and increase your score
* Avoid colliding with the walls or your own tail
* Try to achieve the highest score!
